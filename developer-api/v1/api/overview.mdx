The live Ramp API reference is generated from the [OpenAPI specification](https://docs.ramp.com/openapi/developer-api.json). Plain-text reference: [`llms-api.txt`](https://docs.ramp.com/llms-api.txt).

## Wire Mintlify to OpenAPI

To auto-generate endpoint pages in this repo, add the spec to `docs.json` (local file or supported source) under `api.openapi`, then register generated routes in the **API reference** tab per [Mintlify API documentation](https://mintlify.com/docs/api-playground/overview).

<AccordionGroup>
  <Accordion title="Why start from OpenAPI?">
    Endpoint pages stay aligned with schema changes and reduce manual duplication for parameters and response shapes.
  </Accordion>
  <Accordion title="What to migrate manually?">
    Multi-step workflows, partner policies, and PCI-related disclaimers often stay in MDX guides even when endpoints are generated.
  </Accordion>
</AccordionGroup>
